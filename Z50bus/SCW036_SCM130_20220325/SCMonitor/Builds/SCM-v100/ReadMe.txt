**********************************************************************
** LiNC80                                                           **
**********************************************************************

SCMonitor-v###-L1-LiNC80-32k-ROM
--------------------------------
** Standard build: 16 or 32k ROM
** Includes MS-BASIC with about 47k free memory
** Includes CP/M loader
** Optionally includes Grant Searle loader/monitor (requires 32k ROM)


**********************************************************************
** RC2014                                                           **
**********************************************************************

SCMonitor-v###-R1-RC2014-08k-ROM
--------------------------------
** Standard build: 8k ROM
** All official RC2014 systems can run this
** Paged ROM board set for 8k page size
** 64k RAM board set to start at 8k (not page enabled)

SCMonitor-v###-R2-RC2014-16k-ROM
--------------------------------
** Standard build: 16k ROM
** Paged ROM board set for 16k page size
** 64k RAM board set to start at 16k (not page enabled)
** Includes MS-BASIC with about 47k free memory

SCMonitor-v###-R3-RC2014-32k-ROM
--------------------------------
** Standard build: 32k ROM
** This is the version required if you also want to run CP/M 
** Paged ROM board set for 32k page size
** 64k RAM board set to page enabled
** Includes MS-BASIC with about 31k free memory
** Includes CP/M loader
** Space available in ROM for additional Apps

SCMonitor-v###-R4-RC2014-16k-ROM
--------------------------------
** Standard build: 16k ROM
** This is the version required if you also want to run CP/M 
** Paged ROM board set for 16k page size
** 64k RAM board set to page enabled
** Includes MS-BASIC with about 31k free memory
** Includes CP/M loader

SCMonitor-v###-S1-SC101-16k-ROM
---------------------------------
** Standard build: 32k ROM
** Includes MS-BASIC with about 31k free memory
** Includes CP/M loader

SCMonitor-v###-S2-SC114-32k-ROM
---------------------------------
** Standard build: 32k ROM
** Includes MS-BASIC with about 31k free memory
** Includes CP/M loader
Supports bit-bang serial port of SC114
MiniPro v6.60 checksum = 0x005D 5D8B

SCMonitor-v###-S3-SCS3-32k-ROM
---------------------------------
** Standard build: 32k ROM
** Includes MS-BASIC with about 31k free memory
** Includes CP/M loader
Optimised for SC108 Processor and SC110 Serial
MiniPro v6.60 checksum = 0x005D 0A3E

SCMonitor-v###-S4-Z180-32k-ROM
---------------------------------
** Standard build: 32k ROM
** Includes MS-BASIC with about 31k free memory
** Includes CP/M loader
For Z180 processor using RC2014 serial (not built in serial)
Using MiniPro v6.60 programmer:
v100-S4.0.02 checksum = 0x00DC 78DF (with AT27C512R)

SCMonitor-v###-S5-Z180-32k-ROM
---------------------------------
** Standard build: 32k ROM used
** Includes MS-BASIC with about 31k free memory
** Includes CP/M loader
For Z180 processor using Z180 serial (not RC2014 serial modules)
Using MiniPro v6.60 programmer:
v100-S5.0.03 checksum = 0x07D6 0D79 (with SST39SF040)
